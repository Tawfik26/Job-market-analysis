# Job-market-analysis


This project analyzes a dataset of 32,500+ employees, including names, departments, salaries, job types, and more.  
The analysis covers data cleaning, exploratory data analysis (EDA), salary comparison, and visualizations.

## Features
- Data cleaning & preprocessing  
- Department-wise insights  
- Salary analysis (hourly vs monthly)  
- Outlier detection  

## Tech Stack
Python • Pandas • NumPy • Matplotlib • Seaborn • Jupyter Notebook

## How to Run
pip install -r requirements.txt  
jupyter notebook
